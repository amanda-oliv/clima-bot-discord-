# clima-bot-discord- ☀️
projeto pessoal de um bot de clima feito para o discord, utilizei a api da OpenWeatherMap, foi desenvolvido em pythoh com a biblioteca discord.py

🚀 Como usar
Clone o repositório:

git clone https://github.com/amanda-oliv/clima-bot-discord-.git
cd clima-bot-discord-

Instale as dependências:
pip install -r requirements.txt

Crie um arquivo .env na raiz do projeto com o seguinte conteúdo:
DISCORD_TOKEN=sua_token_aqui
WEATHER_API_KEY=sua_api_key_aqui

Execute o bot:
python bot.py

⚙️ Funcionalidades
Responde com a previsão do tempo quando um usuário digita o comando correspondente.
Fácil de configurar e rodar.
Estrutura simples para aprendizado.
